# **App Name**: Faena Manager 2.0

## Core Features:

- Panel de Control General: Proporciona un panel de control centralizado que muestra los módulos clave: Obras, Operaciones y Prevención.
- Gestión de Obras: Permite la creación y gestión de proyectos de construcción (Obras) con detalles como el nombre del sitio, la dirección y el correo electrónico del cliente.
- Datos Simulados: Utiliza datos simulados para Obras en el MVP inicial, con un camino para integrarse con una base de datos real.
- Módulo de Operaciones (Placeholder): El módulo de Operaciones está en su lugar como marcador de posición para futuras características de operaciones.
- Módulo de Prevención de Riesgos (Placeholder): El módulo de Prevención de Riesgos está en su lugar como marcador de posición para futuras características de gestión de riesgos.

## Style Guidelines:

- Color primario: Azul profundo (#3F51B5) para representar confianza y estabilidad.
- Color de fondo: Azul muy claro (#E8EAF6), casi blanco, para garantizar la legibilidad y una interfaz limpia.
- Color de acento: Violeta (#7E57C2), un color análogo al azul, para llamar la atención sobre elementos importantes de la interfaz de usuario, como los botones primarios.
- Fuente del cuerpo y del título: 'Inter', un sans-serif de estilo grotesco con un aspecto moderno, mecanizado, objetivo y neutral.
- Utilice Tailwind CSS para un diseño receptivo basado en cuadrículas con un ancho máximo de 6xl para el centrado del contenido.
- Emplear iconos geométricos simples para representaciones de módulos en el panel de control.
- Animaciones de transición sutiles para la navegación de páginas y el envío de formularios para mejorar la experiencia del usuario.
